#### 介绍
基于springboot实现的人力资源管理系统，前端采用html+bootstrap，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及企业级项目的入门。


#### 软件架构
前端：html | js | css | jquery | bootstrap  
后端：springboot | mybatis  
环境：jdk1.7 | mysql | maven
